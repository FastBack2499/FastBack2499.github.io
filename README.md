# My Portfolio


Welcome to my personal portfolio website! This site showcases my skills, projects, and experience as a frontend developer and computer engineer. I have built this site using modern web development technologies to provide a smooth and responsive user experience.

## Features

- Technologies Used: 
  - React: For building interactive user interfaces.
  - Next.js: For server-side rendering and SEO-friendly routing.
  - Tailwind CSS: For rapid styling and responsive design.
  
- Projects Section: 
  - Messenger Clone: A full-featured messenger application built with React and Tailwind.
  - React-Tailwind Sample: Demonstrates clean UI design using React and Tailwind.
  
- Responsive Design: The website is fully responsive and adapts to various screen sizes for optimal viewing on desktops, tablets, and mobile devices.
